# 📊 Google Sheets Automation

Скрипт для автоматической работы с Google Sheets через Python. Записывает и читает данные из таблицы.

## 📋 Что делает

- Подключается к Google Sheets через API
- Записывает данные в таблицу
- Читает данные из таблицы

## 🛠 Технологии

- Python 3.11
- gspread
- google-auth

## 🚀 Запуск

1. Установи зависимости:
   pip install gspread google-auth
2. Создай проект в Google Cloud Console
3. Скачай credentials.json и положи рядом со скриптом
4. Открой доступ к таблице для сервисного аккаунта
5. Запусти:
   python3 sheets.py
